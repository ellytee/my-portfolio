# Elton Tinashe Mpofu — Portfolio Website

A personal portfolio website showcasing my skills, services, and data analysis projects.

## Running the site

No build step or dependencies required. It is a static HTML/CSS/JS site.

### Option 1 — Open directly in browser

Double-click `index.html` in your file manager, or run:

```bash
xdg-open index.html
```

### Option 2 — Local development server (recommended)

Using Python (built into Linux/macOS):

```bash
python3 -m http.server 8000
```

Then open **http://localhost:8000** in your browser.

## Project structure

## Tech stack

- HTML5 / CSS3 / JavaScript
- [Typed.js](https://github.com/mattboldt/typed.js/) — typing animation
- [Font Awesome](https://fontawesome.com/) — icons
- [Google Fonts](https://fonts.google.com/) — Ubuntu & Poppins fonts
- jQuery — scroll and menu interactions
